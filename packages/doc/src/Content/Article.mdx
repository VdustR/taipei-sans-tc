## 安裝

```sh
# npm
npm install taipei-sans-tc
# yarn
yarn add taipei-sans-tc
```

## 使用

```js
import 'taipei-sans-tc';
import 'taipei-sans-tc/dist/Light/TaipeiSansTCBeta-Light.css';
import 'taipei-sans-tc/dist/Bold/TaipeiSansTCBeta-Bold.css';
```

## Build

本專案使用 [Font Splitter](https://www.npmjs.com/package/font-splitter) 產出 subsets。

### Requirement

- [Bash](https://www.gnu.org/software/bash/)
- [Docker](https://www.docker.com/)

### Command

```sh
./build.sh
```
